## Quick Answer

**To download your NEWS ROBO project files:**

1. **Look for a Download/Export button** in Figma Make interface (usually top-right corner)
2. **Click "Download Project"** or **"Export to ZIP"**
3. **Save the ZIP file** to your computer
4. **Extract all files** to a folder (e.g., Documents/NEWS-ROBO/)
5. **Open terminal** in that folder
6. **Run:** `npm run wizard`

That's it! The wizard will guide you through building the APK.

---

## Alternative: Use GitHub

If Figma Make has GitHub integration:

1. Click **"Sync to GitHub"** or **"Push to GitHub"**
2. Create/authorize repository
3. On your computer, open terminal and run:
   ```bash
   git clone [your-repository-url]
   cd news-robo
   npm run wizard
   ```
